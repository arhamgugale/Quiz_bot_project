#  Quiz Bot (Tkinter App)

A fun and interactive quiz application built using Python Tkinter GUI.  
Choose from different quiz categories and test your knowledge!



#  Features
- Multiple quiz topics (GK, Science, Fun, Gen Z, etc.)
- User-friendly interface built with Tkinter
- Auto score calculation and results page
- Restart option to replay or choose another quiz
- Uses radio buttons for clean multiple-choice selection



#  Requirements
Make sure you have Python 3.8+ installed.

Then install Tkinter
